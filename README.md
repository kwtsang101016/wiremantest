# 電氣佈線工中工 · 筆試溫習 Web App

建造業議會 **電氣佈線工 中級工藝測試（中工）** 技術知識筆試的溫習與模擬測驗，適合在手機（如 iPhone）上隨時溫習。

## 功能

- **溫習模式**：逐條複習 20 條筆試參考內容（與官方參考資料一致）
- **測驗模式**：模擬筆試，每次隨機 10 題四選一，答對 6 題或以上為合格；交卷後可查看答案對照

## 在 iPhone 上使用

1. **經網頁打開**  
   把整個資料夾放上網頁空間（例如 GitHub Pages、Netlify、或自己的伺服器），用手機瀏覽器打開 `index.html` 的網址。

2. **加到主畫面（當 App 用）**  
   - 在 Safari 打開上述網址  
   - 按底部分享按鈕 →「加入主畫面」  
   - 之後可像 App 一樣從主畫面打開，全螢幕使用

3. **本機預覽（電腦）**  
   在資料夾內用終端機執行：
   ```bash
   npx serve .
   ```
   再用電腦或手機瀏覽器打開顯示的網址（例如 `http://localhost:3000`）。

## 檔案說明

| 檔案 | 說明 |
|------|------|
| `index.html` | 主頁面與各模式版面 |
| `styles.css` | 樣式（手機優先、深色主題） |
| `app.js` | 溫習／測驗流程與計分邏輯 |
| `data.js` | 20 條參考內容與 20 題模擬題庫 |
| `manifest.json` | PWA 設定（名稱、全螢幕等） |
